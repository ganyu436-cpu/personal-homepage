# Personal Homepage

This is a beginner-friendly personal homepage project made with plain HTML, CSS, and JavaScript.

## Files

- `index.html`: page structure
- `styles.css`: colors, layout, and responsive design
- `script.js`: small interactive behavior

## How to run

1. Open the `personal-homepage` folder.
2. Double-click `index.html`.
3. The page will open in your browser.

## What to edit first

Open `index.html` and replace these placeholders:

- `YOUR NAME`
- `your-email@example.com`
- `your-github-name`
- `linkedin.com/in/your-name`
- the About section text

## Suggested learning order

1. Read `index.html` to understand page structure.
2. Read `styles.css` to see how layout and colors work.
3. Read `script.js` to understand the small animation and footer year.

## GitHub upload idea

After you create a GitHub repository, you can upload this folder as your first project and later publish it with GitHub Pages.
